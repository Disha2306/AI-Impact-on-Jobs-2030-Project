# AI-Impact-on-Jobs-2030-Project
A Business Analytics project that explores the impact of Artificial Intelligence on workforce dynamics through interactive Power BI visualizations. The dashboard provides insights into AI replacement risk, future job demand, salary trends, employee satisfaction, and upskilling needs to support data-driven workforce planning
AI Impact on Jobs 2030 – Power BI Dashboard

This project analyzes how Artificial Intelligence is expected to influence jobs, salaries, workforce demand, and automation across various industries. Using a dataset of 3,000+ employee records, the dashboard delivers actionable insights through interactive KPIs, filters, and visualizations, helping organizations identify high-risk roles, workforce trends, and reskilling opportunities.

Key Features
AI Replacement Risk Analysis
Future Demand Forecasting
Salary & Compensation Insights
Employee Satisfaction Analysis
Upskilling & Workforce Readiness Metrics
Interactive Power BI Dashboard with Dynamic Filters
Business Recommendations for Workforce Planning

Transforms raw workforce data into actionable insights that support strategic hiring, reskilling initiatives, and long-term workforce planning in an AI-driven economy.
